### Сборка и запуск

Сборка проекта. Используемый сборщик [Vite]:

npm run build

Выполняется сборка и запуск статического сервера на Express, на порту 3000. http://localhost:3000:

npm run start

Сборка проекта в режиме разработки:

npm run dev
